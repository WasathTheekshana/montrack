## [0.1.0] - 2026-06-01

### Added

- Month-based budget creation with start and end dates
- Budget categories across four types: bills, expenses, savings, and debt
- Transaction logging with income and expense types
- Multi-currency support with exchange rate input; all totals shown in base currency
- Tap-to-toggle on foreign currency transactions to reveal original amount
- Income tracking with expected amounts and payday scheduling
- Daily spending view showing running balance across the month
- Spending breakdown chart by budget category
- Daily trend chart
- Settings page with base currency selector
- Export all data as a JSON backup file
- Import backup to restore all months, budgets, transactions, and income
- Delete all data flow with multi-step confirmation
- Daily reminder notifications with customisable time and creative message rotation
- Test notification button
- Branded loading screen
- Graph paper grid background
- PWA support - installable on desktop and mobile, works offline
- About page
