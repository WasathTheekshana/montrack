# Contributing to "This is for you."

Thank you for taking the time to contribute! Here's how to get started.

## Development setup

```bash
git clone https://github.com/WasathTheekshana/montrack.git
cd montrack
yarn install
yarn dev
```

## Workflow

1. Fork the repo and create a branch from `main`:
   ```bash
   git checkout -b feat/your-feature-name
   ```
2. Make your changes and make sure the project builds:
   ```bash
   yarn build
   yarn lint
   ```
3. Open a pull request against `main` and fill in the template.

## Guidelines

- Keep pull requests focused — one feature or fix per PR.
- Write descriptive commit messages.
- Don't introduce new dependencies without discussion.
- Respect the minimal, pastel design language of the project.

## Reporting bugs

Open an issue using the **Bug report** template and include steps to reproduce.

## Suggesting features

Open an issue using the **Feature request** template. Describe the use case, not just the solution.

## Code of conduct

This project follows the [Contributor Covenant Code of Conduct](./CODE_OF_CONDUCT.md). Be kind.
